# SurgiControl

Sistema inteligente para gestión de instrumental quirúrgico.

## Funciones

- Registro de instrumental
- Gestión de equipos quirúrgicos
- Control de faltantes
- Dashboard interactivo
- Auditoría
- Búsqueda inteligente

## Tecnologías

- Python
- Streamlit
- SQLite
- Pandas

## Ejecución

```bash
streamlit run app.py
```
